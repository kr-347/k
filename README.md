# Krishiv Portfolio Website

A dark multi-page personal portfolio for Krishiv.

## Pages

- `index.html` — home page
- `passions.html` — video editing, forensics/mystery solving, coding/robotics
- `projects.html` — project area with a video placeholder
- `deep-dives.html` — learning topics including toxicology, forensic science, food science, and robotics/coding
- `service.html` — Sandwich Seva through Impact Club and regular NJHS volunteering

## Assets

Keep images and videos inside the `assets` folder.

Current expected files:

```text
assets/profile-photo.jpg
assets/forensics-poster.svg
assets/forensics-video-project.mp4   # optional project video
```

The project video section will show a placeholder until `assets/forensics-video-project.mp4` is uploaded.

## Deploy to Vercel

Upload all files to the root of a GitHub repository, then import that repo into Vercel with:

```text
Application Preset: Other
Root Directory: ./
Build Command: blank
Output Directory: blank
Install Command: blank
```
